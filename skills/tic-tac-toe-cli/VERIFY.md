# Tic-Tac-Toe CLI Verification

## Automated Checks
- Run `pytest` when tests are present; all tests should pass.

## Manual Checks
- Run `python -m tic_tac_toe` and play a short game.
- Confirm invalid input is rejected (non-digit, out of range, occupied cell).
- Confirm the board display updates after each move using positions 1-9.
- Confirm win and draw outcomes are detected and reported.
- Confirm the bot always plays a legal move.
